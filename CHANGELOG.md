# 更新日志

## 1.1.1 - 2026-05-24

- 移植稳定性修复：商店图过大时自动压缩，必要时转为 JPEG，降低群平台发图失败概率。
- 图片发送链路支持 `image/jpeg`，临时文件会按图片类型使用 `.png` 或 `.jpg` 后缀。
- 对齐客户端代理、账号限速、配置注入和请求统计路径等稳定性细节；请求统计数据库写入插件独立数据目录。
- 修复资料图、战绩图发送时的异步调用问题。
- 已用真实凭据在 `temp` 中验证玩家资料图、商店图和强制压缩样张。
