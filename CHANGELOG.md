# 更新日志

## 1.2.1 - 2026-06-27

- 同步上游第三赛季查询配置，大神当前赛季更新为 S23，国服英雄榜赛季更新为 S3，并统一资料、战绩、强度、段位历史和总结等查询的赛季来源。
- 修复 BattleTag 输入中的全角 `＃` 兼容问题，减少复制粘贴导致的玩家搜索失败。
- 同步商店接口 URL 参数变更，补充 `mobileFlow=false`，保持 `/ow 商店` 可用性。
- 优化数据库写入和身份记录链路，继续写入插件独立数据目录，并降低高频查询时的同步写入压力。
- 修复热重载时 API 客户端和后台记录器释放不完整的问题，避免重载后继续复用旧客户端状态。

## 1.2.0 - 2026-05-24

- 新增 `/ow 同玩`、`/ow 同玩详情`、`/ow 同玩锐评`，支持绑定账号省略玩家1，也支持显式传入两个 BattleTag。
- 新增 `/ow 威能 <英雄名>`，查询英雄次级/主要威能、样本数、选取次数、选取率和说明，并输出威能图。
- 新增 `/ow 英雄维基 <英雄名> [问题]`，查询英雄生命值、技能、武器、威能资料；带问题时复用 AstrBot 大模型提供商做资料问答。

## 1.1.1 - 2026-05-24

- 移植稳定性修复：商店图过大时自动压缩，必要时转为 JPEG，降低群平台发图失败概率。
- 图片发送链路支持 `image/jpeg`，临时文件会按图片类型使用 `.png` 或 `.jpg` 后缀。
- 对齐客户端代理、账号限速、配置注入和请求统计路径等稳定性细节；请求统计数据库写入插件独立数据目录。
- 修复资料图、战绩图发送时的异步调用问题。
- 已用真实凭据在 `temp` 中验证玩家资料图、商店图和强制压缩样张。
