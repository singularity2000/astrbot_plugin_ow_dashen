# Modules

`src/modules` 下的所有模组都必须先通过 `tools.py` 进行注册。

注册规范如下：

1. 新增模组前，先在 `src/modules/tools.py` 中登记模组名称和对应入口信息。
2. 注册完成后，再在 `src/modules` 下创建与注册名称对应的文件夹。
3. 文件夹名称必须与 `tools.py` 中注册的模组名称保持一致，避免后续加载、分发和维护时出现歧义。
4. 所有模组资源、实现代码和后续扩展内容，都应放在各自对应名称的文件夹内。

推荐流程：

1. 先确定模组名称。
2. 在 `tools.py` 中完成注册。
3. 创建同名文件夹。
4. 在该文件夹内继续编写模组实现。

示例：

如果要新增 `career` 模组，应先在 `tools.py` 中注册 `career`，然后创建：

```text
src/modules/career/
```

不要跳过注册步骤，也不要先建文件夹后补注册。
